Basic HTML Coding

PART 1 – Creating a Heading

Instruction:
Create a heading using the <h1> tag.

Example:
<h1>Hello ICT!</h1>

PART 2 – Writing a Paragraph

Instruction:
Create a paragraph using the <p> tag.

Example:
<p>ICT is fun and exciting!</p>

PART 3 – Adding an Image

Instruction:
Add an image using <img> with a link.

Example:
<img src="https://tse1.mm.bing.net/th/id/OIP.g8GvZl4kA1u3xtPUBvXGmAHaEK?pid=Api&P=0&h=220" width="200">

PART 4 – Creating a List

Instruction:
Create an unordered list using <ul> and <li>.

Example:
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

PART 5 – Making a Hyperlink

Instruction:
Create a link using <a>.

Example:
<a href="https://www.youtube.com/watch?v=3sur4BmjQt8&list=RD3sur4BmjQt8&start_radio=1">Go to YouTube</a>
